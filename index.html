Api Rest: Node.js, MongoDB e Express

Vamos contruir uma API Restful para criar postagem de blog, com a funcionalidade principal de fazer CRUD, neste projeto vamos utilizar: Node.js, Express e também Mongoose como nosso servidor que vai gerenciar os relacionamentos entre os dados e também a validação dos esquemas entre outras coisas

APIs Restful nos permite dividir o nosso código back-end do código front-end. Podemos utiliza-las em vários aplicativos(mobile ou web)

Usamos Node.js como nossa linguagem de backend, express.js vai nos ajudar a criar rotas mais fáceis e middlewares e usamos mongodb junto com mongoose para criar esquemas e modelos que definem a aparência de nossos dados.
